class mare::thread_storage_ptr

template<typename T, class Allocator = std::allocator<T>>
class mare::thread_storage_ptr< T, Allocator >

Thread-local storage allows sharing of information on a per-task basis, similar as thread-local storage does for threads. A `scheduler_storage_ptr<T>` stores a pointer-to-T (`T*`). In contrast to `task_storage_ptr`, the contents are persistent between tasks. In contrast to `scheduler_storage_ptr`, the contents might be accessed by others while a task is suspended.
