class mare::scheduler_storage_ptr

template<typename T, class Allocator = std::allocator<T>>
class mare::scheduler_storage_ptr< T, Allocator >

Scheduler-local storage allows sharing of information on a per-task basis, similar as thread-local storage does for threads. A `scheduler_storage_ptr<T>` stores a pointer-to-T (`T*`). In contrast to `task_storage_ptr`, the contents are persistent between tasks. In contrast to `thread_storage_ptr`, the contents are guaranteed to be unchanged across context switches. To maintain these guarantees, the runtime system is free to create new objects of type T whenever needed.
